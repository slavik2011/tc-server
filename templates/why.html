Why I created this?

When reading the Terms of Usage of school Chromebooks, I found many security holes that allowed me to create something like this and run it during periods on a school Chromebook.

This site was created by me to show these holes and help to remove them. I like typing, and my purpose was never to create and really use it to type.

If someone from LAUSD sees this site, please don't move it into the blacklist; that would be useless. I can find another hosting and deploy it again.

The source code of this site is currently in a private GitHub repository and ready to share. If LAUSD wants to see my source code, I'll share it on this page.

If any teacher sees this, I'll ask my math teacher Mr. Castaneda to give computer science periods. If not Mr. Castaneda, please take it slowly and let me know. I'll add security to this site to avoid usage for some students.

And please, DON'T USE IT SERIOUSLY; it's just a demonstrational project.

If someone sees this, it means that these security holes are becoming more popular and will also be detected by LAUSD.

Happy learning, slvreal.
